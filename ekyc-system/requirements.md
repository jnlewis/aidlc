# Requirements

## Summary
an application that does eKYC for a banking application

## Personas

For a user:
- able to input documents
- able to input proof of income
- able to input name, address, age, date of birth
- check for liveness during camera capture

For a administrator:
- end goal is that the user is validated
- send for manual review if unable to validate
- able to know my success rate
- knows why the ekyc fail reason
- check for fraud records
